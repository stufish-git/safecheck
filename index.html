Daily Email Summary

Sends an HTML summary of each day's food safety records at 23:59 every night. The trigger must be installed once from Google Apps Script after deploying.

Recipients

Add one email address per line. All addresses will receive the nightly summary.

Apps Script Setup

After updating google-apps-script.js, open Apps Script and run installDailyEmailTrigger() once to schedule the 23:59 trigger. Run removeDailyEmailTrigger() to stop it.

The trigger runs as you (the Apps Script owner), so emails will arrive from your Google account's Gmail address.
